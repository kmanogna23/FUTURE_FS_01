# Manogna Portfolio Website

A modern and responsive personal portfolio website built using HTML, CSS, and JavaScript with a clean dark-themed user interface.

This portfolio website contains multiple sections including a homepage with animated typing effect, an about section describing my interests and career goals, a skills section showcasing my technical abilities, a projects section displaying my web development works, an internship section highlighting my practical experience, and a contact section for communication. The website also includes social media links, resume download option, responsive navigation menu, hover animations, and mobile-friendly design to provide a modern user experience.

## Features

- Modern Dark Theme UI
- Responsive Design
- Smooth Navigation
- Animated Typing Effect
- Skills Section
- About Me Section
- Projects Section
- Internship Section
- Contact Section
- Social Media Links
- Resume Download Option
- Mobile Friendly Layout

## Technologies Used

- HTML5
- CSS3
- JavaScript
